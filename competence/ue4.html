Description du projet

Dans le cadre d'un projet réalisé en binôme, nous avions pour objectif de concevoir et exploiter une base de données contenant les informations de plusieurs étudiants

Le projet comportait plusieurs étapes : concevoir le script de création de la base de données, développer un script de peuplement permettant d'importer des données issues de fichiers CSV tout en vérifiant le format et la validité des données importées, et enfin réaliser des analyses statistiques à partir des données insérées (cette dernière partie étant encore en cours de réalisation).

En binôme, nous avons d'abord modélisé la structure de la base de données en définissant les tables, les relations et les contraintes d'intégrité nécessaires. Nous avons ensuite écrit un script SQL permettant de créer l'ensemble des tables. Une fois la structure en place, nous avons développé un script d'importation de données depuis les fichiers CSV, en intégrant des contrôles de format et de cohérence pour garantir la qualité des données insérées (gestion des dates, des doublons, des contraintes…). Enfin, nous prévoyons d'exploiter ces données via des requêtes d'analyse statistique.

Ce projet m'a permis de consolider mes compétences en modélisation de base de données, en écriture de scripts SQL complexes et en traitement de fichiers CSV. L'analyse statistique des données, qui sera réalisée dans la phase suivante, viendra compléter ce travail par l'exploitation des informations stockées.
